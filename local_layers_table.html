Data layers used in the OHI 2018 global assessment

These data layers were used to calculate the OHI 2018 assessment. Clicking on the layer name in the “Data layer” variable will provide a full description of the data layer (as well as a link to the actual data). The “Description” variable provides links to the scripts (on Github) used to create the data layers (when available).
